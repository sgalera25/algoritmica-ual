# Algorítmica UAL

Este repositorio contiene las soluciones, el material de estudio y los informes de las prácticas realizadas para la asignatura de **Algorítmica** de la **Universidad de Almería (UAL)**.

## Contenido del Repositorio

El repositorio está organizado por prácticas, siguiendo la estructura oficial del curso. A continuación, se detalla el contenido de cada sección:

### Prácticas

* **📁 Práctica 1: Diagramas de flujo y Pseudocódigo**
    * Soluciones y representaciones gráficas de los primeros algoritmos propuestos.
* **📁 Práctica 2: Recursividad**
    * Implementaciones y análisis de algoritmos recursivos frente a sus versiones iterativas.
* **📁 Práctica 3: Análisis Empírico**
    * Estudios sobre el rendimiento de distintos algoritmos en entornos controlados.
    * *Nota: Incluye una sección **OPCIONAL** detallada sobre "Análisis empírico de algoritmos con Java Microbenchmark Harness (JMH)".*
* **📁 Práctica 4: Análisis Teórico**
    * Análisis de complejidad y demostraciones matemáticas sobre la eficiencia de los algoritmos.

## Cómo usar este repositorio

1.  **Clonar el repositorio:** Para tener una copia local, usa el siguiente comando:
    ```bash
    git clone [https://github.com/ualprn519/algoritmica-ual.git](https://github.com/ualprn519/algoritmica-ual.git)
    ```
2.  **Navegar:** Ve a la carpeta de la práctica que desees revisar para ver el código, diagramas e informes.
